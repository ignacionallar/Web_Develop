Sistema Gestor de Horario Docente

Información

Este sistema fue desarrollado para optimizar la asignación y gestión de horarios de profesores en instituciones educativas grandes. Permite a los administradores generar horarios sin conflictos, gestionar la disponibilidad docente y generar reportes detallados.

  • **Objetivo:** Reducir errores manuales en la programación.
  • **Impacto:** Ahorro de tiempo del 30% en la fase de planificación.
Tecnologías empleadas

El proyecto se construyó utilizando un stack moderno para garantizar escalabilidad y rendimiento.

  • **Backend:** PHP (Laravel)
  • **Frontend:** HTML, CSS, JavaScript
  • **Base de Datos:** MySQL
  • **Control de Versiones:** Git / GitHub
